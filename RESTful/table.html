RESTful Routes

A table of all 7 RESTful routes

Name Path HTTP Verb Purpose Mongoose Method
Index /cats GET List all cats cat.find()
New /cats/new GET Show new cat form N/A
Create /cats POST Create a new cat, then redirect somewhere cat.create()
Show /cats/:id GET Show info about one specific cat cat.findById()
Edit /cats/:id/edit GET Show edit form for one cat cat.findById()
Update /cats/:id PUT Update particular cat, then redirect somewhere cat.findByIdAndUpdate()
Destroy /cats/:id DELETE Delete a particular cat, then redirect somewhere cat.findByIdAndRemove()
